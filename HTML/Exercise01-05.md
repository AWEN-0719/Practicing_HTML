# 小結 練習 01 ~ 05 
## HTML / CSS / Bootstrap 練習筆記（前端基礎）
這是個人前端基礎學習紀錄，內容涵蓋 HTML 結構、CSS 版面配置、CSS Grid 與 Bootstrap 5
目的在於建立正確的前端基礎觀念，並作為可檢視的學習成果

### 目標
- 熟悉 HTML 語意化結構
- 理解 CSS 基本排版與版型控制
- 掌握 CSS Grid 的實務用法
- 學會使用 Bootstrap 快速建立 RWD 與常見 UI 元件

#### 使用技術（Tech Stack）
- HTML5
- CSS3
- CSS Grid
- Bootstrap 5

|  練習  |  主題  |	重點能力  |

### 練習 01｜HTML 基本結構
#### 學習目標
- 建立正確的 HTML 文件結構
- 理解區塊與內容的關係
#### 重點整理
- 使用 header、main、footer 區分版面
- HTML 負責結構，不處理外觀
#### 對應能力
- 能閱讀與撰寫結構清楚的 HTML
----

### 練習 02｜CSS 樣式與排版基礎
#### 學習目標
- 使用 CSS 控制字型、顏色與間距
#### 重點整理
- box model（margin / padding / border）
- class 與選擇器的使用
#### 對應能力
- 能調整現有版型樣式
----

### 練習 03｜版型配置練習
#### 學習目標
- 建立多區塊版型
#### 重點整理
- 區塊配置邏輯
- 視覺層次的建立
#### 對應能力
- 能依設計稿切出基本版型
----

### 練習 04｜CSS Grid 進階版型
#### 學習目標
使用 CSS Grid 建立精準版面
#### 重點整理
- ```grid-template-columns``` / ```rows```
- ```grid-column``` / ```grid-row```
- 偽元素與層級控制（z-index）
#### 對應能力
- 能處理複雜設計稿
- 了解現代 CSS 版型工具
----

### 練習 05｜Bootstrap 與 RWD
#### 學習重點
範例一：單頁式 RWD 版型
- 使用 Bootstrap Grid 快速完成版型
- 響應式設計（RWD）
範例二：Bootstrap 元件整合
- 使用元件
- Navbar（含漢堡選單）
- Accordion
- Table
- Grid System
學習重點
- 不寫 JavaScript 即可完成互動
- Bootstrap 適合快速原型與後台系統
#### 對應能力
- 能使用 Bootstrap 快速完成專案
- 理解框架與原生 CSS 的差異
----

### 學習總結
- 練習 HTML / CSS 基礎
- 使用 CSS Grid 與 Bootstrap 處理不同需求
- 理解何時該使用框架、何時該自訂樣式
